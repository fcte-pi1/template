# _Mecânica e Estruturas_

Esta pasta deverá ser utilizada para armazenar arquivos relativos ao projeto estrutural, mecanismos, materiais, estudos aerodinâmicos, equações dinâmicas, construção e manufatura, impressão 3D, e correlatos.

Os arquivos a serem incluídos, são, dentre outros:
- Arquivos CAD de peças e montagens (FCStd, STEP, etc). 
- Desenhos técnicos (pdf, etc).
- Arquivos de impressão 3D (STL, etc).
- Imagens rasterizadas e vetorizadas (eps, pdf, svg, etc).
- Documentos e estudos realizados para o projeto (markdown, LaTeX, etc).

Durante o projeto, os grupos são encorajados a realizarem estudos para o projeto de estruturas, mecanismos, aerodinamica, dentre outros, e muitos desses estudos não caberão no relatório a ser entregue. Esses estudos podem ser colocados também nessa pasta, preferencialmente utilizando markdown (.md) para facilitar a leitura em ambiente web. Lembre-se que você pode utilizar equações em formato LaTeX no markdown e não deixe de utilizar links para o material de referencia que utilizaram. Estudos bem realizados contarão de maneira positiva na avaliação das equipes, porém lembre-se que não é permitida a utilização de IA, e que esse material deve ser principalmente útil à equipe.

> [!WARNING]
> Não acrecente arquivos da parte de hardware eletrônico aqui, como arquivos de KiCAD. Estes devem ser salvos na pasta [hw](https://github.com/fcte-pi1/template/tree/main/hw) deste repositório.
