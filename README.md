# Template PI1

Esse é o template de repositório para ser utilizado pelos grupos de PI1 para organizar seu projeto. O template é dividido em pastas onde cada parte do projeto deve ser armazenada. Os arquivos a serem armazenados incluem documentação, código, arquivos de CAD, e dados. 

Lembrem sempre de evitar enviar arquivos muito grandes (>5MB). No caso de vídeos e outros arquivos pesados que são necessários, armazenar o arquivo em outra plataforma e colocar aqui apenas o link.

A estrutura de pastas do projeto não reflete a divisão de equipes. Os membros podem trabalhar nas diferentes pastas dependentendo da necessidade do projeto.
